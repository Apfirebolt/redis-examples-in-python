# A GUI application using Redis as database

This is a GUI application using Redis as database. It is a simple application for saving and retrieving data from Redis.

This uses a sample dataset from myntra.com - a popular apparel store in India. Co-pilot AI completion tool was used in making this project.

## Features

* Save data into Redis
* Retrieve data from Redis
* Delete data from Redis

## Requirements

* Python 3.6 or above
* Redis
